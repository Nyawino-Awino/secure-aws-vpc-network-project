# 🔐 Secure AWS VPC Network Lab

## Overview
This project demonstrates a secure multi-tier AWS network architecture using VPC, subnets, security groups, and routing configurations.

---

## Architecture
- Public subnet (Web server)
- Private subnet (Database)
- Internet Gateway
- NAT Gateway
- Security Groups & NACLs

---

## AWS Services Used
- VPC
- EC2
- IAM
- Security Groups
- NACLs
- CloudWatch (optional)

---

## Objectives
- Build secure network segmentation
- Configure firewall rules
- Implement routing for public/private subnets
- Practice troubleshooting network issues

---

## Key Learnings
- TCP/IP networking in AWS
- Firewall configuration (Security Groups & NACLs)
- Network troubleshooting
- Cloud security fundamentals
